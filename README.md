# Projeto DevOps - Ambiente Multi-Container (Flask + PostgreSQL)

Este projeto foi desenvolvido para a disciplina de DevOps com o objetivo de praticar:

- Criação de `Dockerfile` multi-stage usando imagem `alpine`;
- Configuração de ambiente multi-container com `docker-compose`;
- Uso de volumes para persistência de dados;
- Criação de rede customizada para comunicação entre containers;
- Uso de variáveis de ambiente para configuração da aplicação;
- Criação de usuário de banco separado do usuário root.
